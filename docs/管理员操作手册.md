# 忍法帖计算器管理员操作手册

## 记住一句话

以后日常更新只改云开发数据库里的这一条记录：

```text
云开发 -> 数据库 -> naruto_pass_config -> _id: global
```

不要新增记录，只改 `global` 这一条。

---

## 一、发布小程序

在微信开发者工具里：

1. 点「编译」，确认页面正常。
2. 点「上传」。
3. 版本备注可以写：

```text
支持云端配置赛季、S忍、秘卷和背景图
```

4. 打开小程序后台 -> 版本管理。
5. 可以先设为体验版，体验版页面路径填：

```text
pages/index/index
```

确认没问题后再提交审核。

---

## 二、修改赛季结束时间

进入：

```text
云开发 -> 数据库 -> naruto_pass_config -> _id: global
```

修改这两个字段：

```text
seasonEndDate = 2026-07-31
seasonEndTime = 12:00
```

格式要求：

```text
日期：YYYY-MM-DD
时间：HH:mm
```

示例：

```text
2026-07-31
12:00
```

---

## 三、上传图片

进入：

```text
云开发 -> 存储 -> 上传文件
```

上传头像、秘卷图标或背景图。

上传后复制图片的 `File ID`，格式类似：

```text
cloud://cloud1-d6gy16a3xf0e1a5c4/images/xxx.png
```

你的环境 ID 是：

```text
cloud1-d6gy16a3xf0e1a5c4
```

---

## 四、修改蒲式头像

在 `global` 记录里修改字段：

```text
urashikiImage
```

值填云存储图片 File ID：

```text
cloud://cloud1-d6gy16a3xf0e1a5c4/images/urashiki.jpg
```

## 五、修改首页背景图

在 `global` 记录里修改字段：

```text
backgroundImage
```

值填云存储图片 File ID：

```text
cloud://cloud1-d6gy16a3xf0e1a5c4/images/background.png
```

## 六、添加新 S 忍

在 `global` 记录里添加或修改字段：

```text
extraNinjas
```

字段类型选：

```text
string
```

值粘贴下面这一整段：

```json
[
  {
    "id": "new-pass-s-202607",
    "name": "新S忍名字",
    "alias": "短名",
    "era": "new",
    "image": "cloud://cloud1-d6gy16a3xf0e1a5c4/images/new-pass-s.jpg"
  }
]
```
字段说明：

```text
id    用英文、数字、横杠，不要用中文
name  用户看到的中文名
alias 头像下面显示的短名，可以不填
era   价格规则
image 云存储图片 File ID，可以不填
```

`era` 的规则：

```text
new = 新 S，15 币/片
old = 老 S，前 80 片 25 币，后 20 片 50 币
```

没有头像时，可以先这样填：

```json
[
  {
    "id": "new-pass-s-202607",
    "name": "新S忍名字",
    "era": "new"
  }
]
```

## 七、添加新秘卷

在 `global` 记录里添加或修改字段：

```text
extraScrolls
```

字段类型选：

```text
string
```

值粘贴下面这一整段：

```json
[
  {
    "id": "new-scroll-202607",
    "name": "新秘卷名字",
    "image": "cloud://cloud1-d6gy16a3xf0e1a5c4/images/new-scroll.png"
  }
]
```

没有图标时，可以先这样填：

```json
[
  {
    "id": "new-scroll-202607",
    "name": "新秘卷名字"
  }
]
```

---

## 八、一次添加多个

多个 S 忍或秘卷都放在同一个数组里，每个对象之间用英文逗号。

示例：

```json
[
  {
    "id": "s-202607",
    "name": "第一个S",
    "era": "new"
  },
  {
    "id": "s-202609",
    "name": "第二个S",
    "era": "new"
  }
]
```

注意：

```text
英文逗号：,
英文引号："
不要用中文逗号：，
不要用中文引号：“ ”
```

---

## 九、日常更新流程

每次新忍法帖更新后，按这个顺序做：

1. 上传新 S 忍头像和新秘卷图标到云存储。
2. 复制图片 File ID。
3. 打开数据库 `naruto_pass_config`。
4. 选择 `_id: global`。
5. 修改 `seasonEndDate` 和 `seasonEndTime`。
6. 修改 `extraNinjas`。
7. 修改 `extraScrolls`。
8. 保存后重新打开小程序查看。

## 十、常见错误

### 1. 新内容没出现

先确认：

```text
字段名是不是 extraNinjas 或 extraScrolls
字段类型是不是 string
JSON 里是不是英文逗号和英文引号
小程序是不是重新打开了
```

### 2. 图片不显示

先确认：

```text
image 填的是云存储 File ID
File ID 以 cloud:// 开头
图片没有被删除
```

### 3. JSON 填错了

最常见是少逗号、少引号。

正确格式：

```json
[
  {
    "id": "example",
    "name": "示例",
    "era": "new"
  }
]
```
