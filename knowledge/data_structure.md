# Naruto Tool Knowledge Base

## Purpose
Store source material for the Naruto Mobile battle-pass calculator, including rule notes, season schedules, screenshots, Excel sheets, and PDF references.

## Files
- `../docs/rules.md` - Known calculation rules for ninja fragments, secret scroll fragments, exchange coins, and pass levels.
- `../docs/architecture.md` - Project architecture notes.
- `../docs/frontend-guide.md` - Frontend editing and design notes.
- Add new PDFs, Excel sheets, or Markdown notes here when collecting future rules.

## Coverage
This knowledge base currently covers the local project documentation and can be extended with official or community-maintained rule references.
